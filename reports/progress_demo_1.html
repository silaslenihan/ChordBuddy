Data

The data I'm using in this project current comes from myself. The API I'm using works on the frontend, and does not actually return data to me, rather, I provide data to tell it what to display. The current data structure I am operating under is a list of JSON objects. Each JSON object contains information about a musical key, and the chords listen in that key. I pass that data to the frontend components which use the scales_chords_api to display that list in chord form, as well as allow the user to play the sound the chord makes. Here is an example of the data structure:
json_data = { "C": { "chords": ["Cmaj","Dm","Em","Fmaj","Gmaj","Am","Bdim7"] }, "D": { "chords": ["Dmaj","Em","Fm","Gmaj","Amaj","Bm","Cdim7"] }, "E": { "chords": ["Emaj","Fm","Gm","Amaj","Bmaj","Cm","Ddim7"] }, "F": { "chords": ["Fmaj","Gm","Am","Bmaj","Cmaj","Dm","Edim7"] } } Right now, this is only a sample of the data and when the full dataset is done, it will have 12 scales total with probably 10-20 chords in each.
